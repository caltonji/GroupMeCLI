# GroupMeCLI
GroupMe command line interface
## Installation
npm install groupme
## Usage
####groupme groups
####groupme use <groupid>
####groupme messages
####groupme send <message of whatever length to current group>
####groupme like -id <id of message>
####groupme -watch 
to switch to watching the current group

###In watch mode
####like <id>
####send <message of whatever length>
####ctrl-c to exit
